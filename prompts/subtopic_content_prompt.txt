# Variables, Data Types, and Operators in Python

## Introduction
Variables are the foundation of any programming language. Think of them as labeled containers that store information. Just like you might label kitchen containers to remember what's inside (flour, sugar, rice), variables help your computer remember and organize data. In this lesson, we'll explore how Python handles variables, the different types of data you can store, and how to perform operations on them.

## What Are Variables?
A variable in Python is a named location in computer memory where you can store data. Unlike some programming languages, Python is dynamically typed, meaning you don't need to declare what type of data a variable will hold - Python figures it out automatically.

Creating a variable is simple:
```python
name = "Alice"
age = 25
height = 5.6
is_student = True
```

Notice we didn't tell Python that 'name' is text, 'age' is a number, or 'is_student' is a true/false value. Python intelligently determined this based on the values we assigned.

## Understanding Data Types
Python has several built-in data types. The most common ones are:

**1. Strings (str)**: Text data enclosed in quotes
- Example: "Hello World", 'Python', """Multi-line text"""
- Used for: Names, messages, any textual information

**2. Integers (int)**: Whole numbers without decimals
- Example: 42, -17, 1000
- Used for: Counting, indexing, discrete quantities

**3. Floats (float)**: Numbers with decimal points
- Example: 3.14, -0.5, 2.0
- Used for: Measurements, prices, scientific calculations

**4. Booleans (bool)**: True or False values
- Example: True, False
- Used for: Conditions, flags, logic operations

## Working with Operators
Operators are symbols that perform operations on variables and values.

**Arithmetic Operators:**
- Addition (+): 5 + 3 = 8
- Subtraction (-): 10 - 4 = 6
- Multiplication (*): 6 * 7 = 42
- Division (/): 15 / 3 = 5.0
- Integer Division (//): 15 // 4 = 3
- Modulus (%): 15 % 4 = 3
- Exponentiation (**): 2 ** 3 = 8

**Comparison Operators:**
- Equal to (==): 5 == 5 returns True
- Not equal (!=): 5 != 3 returns True
- Greater than (>), Less than (<)
- Greater than or equal (>=), Less than or equal (<=)

**Real-World Example:**
Let's calculate a simple shopping bill:
```python
apple_price = 0.5
apple_quantity = 6
banana_price = 0.3
banana_quantity = 10

total_apples = apple_price * apple_quantity
total_bananas = banana_price * banana_quantity
grand_total = total_apples + total_bananas

print(f"Your total bill is: ${{grand_total}}")
```

## Key Takeaways
✓ Variables store data and don't require type declaration in Python
✓ Main data types: strings (text), integers (whole numbers), floats (decimals), booleans (True/False)
✓ Use arithmetic operators for calculations
✓ Comparison operators help make decisions in your code
✓ Python is beginner-friendly with intuitive syntax
